# 基础语法

- 注释

- 变量

  - ```
    程序中，数据都是临时存储在内存中，为了了更更快速的查找或使⽤用这个数据，通常我们把这个数据在内存中存储之后定义一个名称，这个名称就是变量
    变量就是一个存储数据的的时候当前数据所在的内存地址的名字
    ```

  - 变量命名

- Debug

- 数据类型

  - 数值： int/float
  - 布尔型：True/ False
  - str（字符串）
  - list (数组)
  - tuple （元祖）
  - set （集合）
  - dict （字典）

- 检测数据类型

  - type()

- 输出

  - 格式化输出

    - 格式化符号

      >%s
      >
      >%d
      >
      >%f

    - f格式化字符串

      ```
      f('{}')
      ```

    - 转义字符

      ```
      \n  换行
      \t	制表符， 一个tab 四个空格的距离
      ```

    - print结束符

      ```
      print('内容', end="")
      ```

- 输入

  ```
  input("提示信息")
  ```

- 数据类型转换

  ```
  int(x[, base])
  float(x)
  str(x)
  eval(str) // 将内容转换成有效的python表达式
  tuple(x) // 将一个序列转换成元祖
  list(x) // 将一个序列转换成数组
  ```
